Разработан плагин **USM Notes**, который добавляет систему заметок с:

* фильтрацией через шорткод

---

## ⚙️ Установка и запуск

1. Скопировать папку `usm-notes` в:

   ```
   wp-content/plugins/
   ```
2. Перейти в админку WordPress
3. Активировать плагин **USM Notes**

## ❓ Ответы на контрольные вопросы

### 1. Таксономия vs метаполя

**Таксономия:**

* используется для группировки (категории, теги)
* пример: приоритет (high/medium/low)

**Метаполе:**

* хранит дополнительные данные
* пример: дата напоминания

---

### 2. Зачем nonce?

Nonce защищает от:

* CSRF атак
* несанкционированного изменения данных

Если не проверять:
→ злоумышленник может отправить POST-запрос и изменить данные

---

### 3. Важные аргументы register_post_type()

* `public` — влияет на доступность на фронтенде
* `has_archive` — наличие страницы архива
* `supports` — какие поля доступны
* `labels` — UX в админке
